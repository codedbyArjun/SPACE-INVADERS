# 👾 Space Invaders

This is a simple Space Invaders game that I built using Python as part of Dr. Angela Yu's **100 Days of Code** course.

I wanted to build something that was more than just a small practice program, and this project helped me understand how different Python concepts come together to create a complete game.

## Features

- Move the player left and right
- Shoot bullets
- Multiple enemies
- Collision detection
- Score counter
- Game Over screen

## Built With

- Python
- Turtle Graphics

## What I Learned

Working on this project helped me get more comfortable with:

- Object-Oriented Programming
- Functions
- Loops
- Event handling
- Collision detection
- Breaking a project into multiple files

It also taught me that debugging is a big part of programming 😅.

## How to Run

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Space-Invaders.git
```

Go into the project folder:

```bash
cd Space-Invaders
```

Run the game:

```bash
python main.py
```
